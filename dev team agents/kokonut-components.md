# KokonutUI Pro Component Catalog

> **Reference for Frontend & UX Agents**  
> Use this catalog to select high-quality, pre-built components for the Phase 4 UI Overhaul.  
> **Install**: Run the install command in the `frontend/` root (or specific package if needed, but usually shared).  
> **Pattern**: Install to `@datalens/shared/src/components/kokonut/` (or similar) to keep them organized.
